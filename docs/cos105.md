# COS105 — 되감기는 문

## 개요

**COS105**는 통과한 사람의 이동 순서를 부분적으로 되감는 문 또는 문 형태의 공간 이상이다. 물리적인 문짝이 존재하지 않아도, 두 지점 사이에 ‘문턱’이 형성되면 COS105로 분류한다.

## 특성

- 출입자는 들어가기 전의 위치로 되돌아왔다고 느끼지만, 시계는 몇 분에서 몇 시간 뒤를 가리킨다.
- 문턱을 반복 통과하면 같은 대화를 다른 순서로 듣게 된다.
- 감시 카메라에는 통과 장면이 역재생처럼 기록된다.

## COS100과의 연관성

[COS100 — 회귀하는 공백핵](./cos100.md)의 색인 회귀 효과는 COS105의 공간 되감기와 상호 증폭된다. 두 현상이 같은 시설 내에 존재할 경우, 문서 작성 시각과 실제 출입 시각이 서로 맞지 않으며 기록 순서가 역전된다.

## 취급 절차

COS105 문턱 양측에는 동기화되지 않은 두 개의 시계를 설치한다. 두 시계가 동시에 같은 시각을 가리키면 통행을 중지한다.
